# Nimble Gravity API Challenge

Mini aplicación en React + TypeScript que consume la API de Nimble Gravity para mostrar posiciones abiertas y enviar postulaciones.

## Tech Stack

- **React 19** + **TypeScript**
- **Vite** como bundler y dev server

## Getting Started

```bash
# Instalar dependencias
npm install

# Iniciar el servidor de desarrollo
npm run dev
```
